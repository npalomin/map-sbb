New York Data

nybbox.bb

- Shape used for New York bounding box.

way_kerb_tags.csv

- Tags applied to Way OSM entries that intersect the Open Data New York sidewalk polygons

ny_kerb_points.shp

- Data returned from OSM query for Kerbs. Individual coordinate geometries.

ny_kerb_lines.shp

- Data returned from OSM query for Kerbs. LineString geometries formed by joining coordinates.
