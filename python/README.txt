Python Code for OSM Queries and Analysis

london_osm_pavement_nodes.ipynb, london_osm_pavement_nodes.html

- Query OSM for Node entries that intersect either pavement or carriadgeway polygons in Clapham Common, London. Used to look at what tags are used to describe features in pedestrian and vehicle road space.


osm_streetspace_utils.py

- Small library of functions for querying OSM data, forming geometries from returned data, and aggregating metadata

ny_osm_kerb_tags.py

- Script investigating possible tags used to identify kerb lines

ny_osm_nodes.py

- Script to get Points and LineStrings representing kerb lines in New York
